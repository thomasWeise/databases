In our excursion on the available \pgls{dbms} in \cref{sec:software}, we mentioned that the open source office suite \libreoffice\ provides program named \libreofficeBase.
\libreofficeBase\ can be used as either a stand-alone \pgls{DBMS} or as a nice user interface that can connect to other \pglspl{db}~\cite{FNFHWSKLSSGLFRSRPLJG2022BG7R1BOL7C,S2022L7PFEUU}.
It has a functionality similar to Microsoft Access~\cite{SSI2023MA2BTA,B2020HOMA2,UC2021AFD}, but it is free.
We will therefore use this software to connect to \postgresql.
Let us therefore discuss how \libreoffice~\cite{DF2024LTDF,GL2012LTSOOSSCBAFACSOL,S2022L7PFEUU} can be installed.%

\hsection{Installing yEd}%
\label{sec:installingYed}%
%
\yEd\ is a free and platform-independent editor for graph data~\cite{SG2015MDAWY,Y2011YGEM}.
It is useful when we learn about \pglspl{ERD} when modelling the conceptual schema of a \db, which we will discuss in \cref{sec:conceptualSchemaDesign}~(and where we will also use \yEd).
While it is not \glsreset{OSS}\pgls{OSS}, \yEd\ is free and works on each \pgls{OS} for which \pgls{Java} is available.%
